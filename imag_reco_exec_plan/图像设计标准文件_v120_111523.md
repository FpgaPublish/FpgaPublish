### 参数设计
* 参数隐藏在参数页中
* 参数要求全部结构体化
* 参数的类型需要进行统一
* 这个交由控件实现
```
|---------------------------------------------------|
| file | mode | connect | param | help |   
|---------------------------------------------------|
标准参数页选项
```

* 需要保持固定格式


### 代码规范
* 列举所有可能的内容
* 按照使用方式进行布局，确保某类数据可以在确定的位置获取
* 得到整个设计的设计流标准
* 进行代码设计 （标准同步到代码规范中去）
